Available Courses

Course Code: IT101

Course Description:

This course introduces fundamental data structures and algorithms used in computer science.

Topics Covered:

  • Arrays and Linked Lists
  • Stacks and Queues
  • Trees and Graphs
  • Sorting and Searching Algorithms
  • Time Complexity Analysis

Assessment:

  • Mid-term Exam: 30%
  • Final Exam: 40%
  • Assignments: 20%
  • Project: 10%

Course Code: IT201

Course Description:

Learn modern web development technologies and best practices.

Topics Covered:

  • HTML5 and CSS3
  • JavaScript and ES6
  • React.js Framework
  • Node.js and Express
  • Database Integration

Assessment:

  • Mid-term Exam: 25%
  • Final Exam: 35%
  • Projects: 30%
  • Lab Work: 10%

Course Code: IT301

Course Description:

Comprehensive study of database design, implementation, and management.

Topics Covered:

  • Relational Database Design
  • SQL Programming
  • Normalization
  • NoSQL Databases
  • Database Security

Assessment:

  • Mid-term Exam: 30%
  • Final Exam: 40%
  • Database Project: 20%
  • SQL Assignments: 10%
